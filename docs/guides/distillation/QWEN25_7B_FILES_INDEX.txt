ORDRE D'EXÉCUTION RECOMMANDÉ
================================================================================

Phase 1: LECTURE (10 minutes)
  1. cat LISEZMOI_QWEN25_7B.md        # Vue d'ensemble FR
  2. cat QWEN25_7B_QUICKSTART.md      # Instructions rapides

Phase 2: SETUP (5 minutes)
  3. ./setup_qwen25_7b_env.sh         # Configuration environnement

Phase 3: CONFIGURATION (2 minutes)
  4. nano distill_qwen25_7b.py        # Éditer device="cpu"

Phase 4: DISTILLATION (10-20 heures)
  5. screen -S distill
  6. ./run_qwen25_7b_distillation.sh  # Lancer distillation
  7. Ctrl+A + D                        # Détacher screen

Phase 5: VALIDATION (7 minutes, le lendemain)
  8. screen -r distill                 # Vérifier logs
  9. ./test_qwen25_7b_model.sh        # Tests
  10. ./evaluate_qwen25_7b.sh         # Évaluation

Phase 6: DÉPLOIEMENT (10 minutes, si score ≥ 91%)
  11. ./deploy_qwen25_7b.sh           # Déploiement production
